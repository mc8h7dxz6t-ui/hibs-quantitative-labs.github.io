Forensic Media Suite

Gold-standard YouTube → media conversion on Apple Silicon and Linux. Zero-copy RAM pipes, hardware VideoToolbox encoders, HDR/5.1 preservation, and SHA-256 chain-of-custody logging.

Stack

  • yt-dlp — stream extraction & metadata probe
  • FFmpeg — h264_videotoolbox, hevc_videotoolbox, aac_at on macOS
  • Python — subprocess orchestration, no wrapper overhead

Install (macOS M-series)

brew install ffmpeg
pip install -r requirements.txt
python m5_forensic_media_suite.py doctor

Usage

# Single convert
python m5_forensic_media_suite.py convert "YOUTUBE_URL" -f mp4

# Playlist batch
python m5_forensic_media_suite.py batch "PLAYLIST_URL" -f mp3

# Watch-folder dashboard
python m5_forensic_media_suite.py watch
